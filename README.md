**Cuerpo de Bombero del Cantón Colimes**

Sistema de Permisos Para el **Cuerpo de Bomberos de Colimes**

## Contenido

1. Generar Permisos
2. Consultar Permisos por Fecha y/o Tipo de Permisos
3. Consultar Permisos por Nombre de Cliente y/o Cédula/RUC 
4. Visualizar Permisos existentes
5. Agregar Tipos de Permisos
6. Editar Tipos de Permisos

## Instalación

1. Descargar o Clonar el repositorio
2. Instalar MySQL
3. Ejecutar el script `cbc.sql`
4. Usuario y Contraseña estan en el script
5. Correr el programa
